Sobre mim

Meu nome é Maykom Mykael Wochniscki e sou apaixonado por tecnologia e programação. Sempre tive curiosidade em entender como as coisas funcionam “por trás das telas” e, ao mergulhar no universo da programação, descobri não apenas uma área de estudo, mas também um espaço onde posso transformar ideias em realidade. Sou uma pessoa curiosa, dedicada e que gosta de aprender de forma prática. Acredito que a melhor forma de evoluir é colocar a mão no código, errar, testar e, a partir disso, encontrar soluções cada vez melhores.

Dentro das aulas e estudos, venho criando projetos práticos que me ajudam a aplicar o que aprendo: Pequenos sistemas em Python, explorando estruturas de repetição, funções, manipulação de arquivos e até conceitos de orientação a objetos. Sites e páginas web, aplicando HTML, CSS e JavaScript para praticar design responsivo e interatividade. Consultas em SQL, utilizando joins e relacionamentos entre tabelas para organizar e recuperar informações de forma estruturada. Projetos compartilhados no GitHub, servindo como registro da minha evolução e como vitrine do meu aprendizado. Esses projetos não são apenas exercícios, mas também uma forma de contar minha história de aprendizado e mostrar meu crescimento passo a passo.
